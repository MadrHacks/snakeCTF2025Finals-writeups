## Solution

Challenge shows a screenshot from Google Maps. Heading is visible, as well as capture date. We also see that the area has not been captured in any other dates. We can also see the place is in the FVG Autonomous Region of Italy.
Go [here](https://map-degen.vercel.app/). Filter for August 2011, set correct heading, limit to FVG area. Use any tool to compare resulting images for a match (or do it manually).
Solution is: 46.2807885, 13.2265755
